DANMU ANALYZER

Word Cloud

pattern: {{"http://{host}/word-cloud/{platform}-{roomid}?buffer={number}&nn={string1}&nn={string2}"}}

platform: douyu supported

roomid: room id

bufferTime(unit: s): time of danmu stream buffer, min value is 30

nn: filter by nicknames of danmus

e.g.: http:xxxx/word-cloud/douyu-229346?bufferTime=30
